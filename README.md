# 🍴 Restoran Məlumatlarının Analizi (Python)

## 📌 Layihə Haqqında
Bu layihə restoran əməliyyatlarının və müştəri davranışlarının Python vasitəsilə həyata keçirilmiş geniş analizidir. Analiz zamanı 244 fərqli sifariş məlumatı əsasında **Tip (bəxşiş)** meylləri və gəlirlilik göstəriciləri araşdırılmışdır.

## 📊 Əsas Nəticələr
* **Ən gəlirli gün:** Həftə sonları, xüsusilə Şənbə (Saturday) günü ən yüksək gəlir və **Tip** daxil olan gündür.
* **Tip tendensiyası:** Ümumi hesab məbləği artdıqca, müştərilərin verdiyi **Tip** miqdarı da mütənasib şəkildə artır.
* **Müştəri profili:** Cinsiyyət və siqaret çəkmə vərdişlərinə görə xərcləmə modelləri müqayisə edilmiş və statistik fərqlər qeyd olunmuşdur.

## 🛠 İstifadə Olunan Kitabxanalar
* **Pandas & NumPy** - Məlumatların manipulyasiyası və təmizlənməsi.
* **Seaborn & Matplotlib** - Statistik qrafiklərin və vizuallaşdırmanın hazırlanması.

## 📁 Dataset haqqında məlumat
* **Mənbə:** Seaborn "tips" dataset (Kaggle).
* **Həcm:** 244 sətir, 12 sütun (Analiz prosesində yeni xüsusiyyətlər/metriklər əlavə edilmişdir).
* **Əsas Sütunlar:** total_bill, tip, sex, smoker, day, time, size və hesablanmış digər xüsusiyyətlər.
